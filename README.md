# 随机专家抽取器

这是一个基于 Python 的 Windows GUI 程序，用于从 Excel (.xlsx) 文件中筛选专家并按条件随机抽取。

## 功能特点

- 选择 Excel 文件并自动识别表头
- 支持最多 5 组筛选条件
- 支持按学历、职称/职称层级、申报专业、推荐部门和所需人数进行筛选
- 支持排除多个工作单位，并在输入时提供联想提示
- 支持将筛选结果导出为 CSV，并包含生成时间、筛选条件和排除单位信息
- 默认全屏显示，界面风格贴近 Windows 11

## 文件结构

- expert_selector.py：主程序
- requirements.txt：依赖清单
- tests/test_filter_logic.py：筛选逻辑测试

## 运行方式

1. 安装 Python 3.9+。
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 运行程序：

```bash
python expert_selector.py
```

## Excel 文件要求

表头必须包含以下列：

- 序号
- 姓名
- 性别
- 出生年月
- 年龄
- 学历
- 工作单位
- 职务/职业资格
- 职称/职称层级
- 手机号码
- 申报专业
- 业绩成果
- 推荐部门
- 备注

## 使用说明

1. 点击“选择 Excel 文件”导入数据。
2. 在“筛选条件”中设置一组或多组筛选条件，最多 5 组。
3. 在“需排除单位”中输入想要排除的工作单位，输入时会自动提示匹配项。
4. 点击“筛选”生成结果。
5. 点击“导出”将结果保存为 CSV。
